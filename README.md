## 🧮 SGPA Formula

The SGPA is calculated using the formula:

SGPA = Σ (Ci × Pi) / Σ Ci

Where:

| Symbol | Meaning |
|------|------|
| \(C_i\) | Credits of the *i-th* subject |
| \(P_i\) | Grade points of the *i-th* subject |
